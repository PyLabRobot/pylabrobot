pylabrobot.resources.Tube#

class pylabrobot.resources.Tube(name: str, size_x: float, size_y: float, size_z: float, max_volume: float, material_z_thickness: float | None = None, category: str = 'tube', model: str | None = None, bottom_type: TubeBottomType | str = TubeBottomType.UNKNOWN, compute_volume_from_height: Callable[[float], float] | None = None, compute_height_from_volume: Callable[[float], float] | None = None, height_volume_data: Dict[float, float] | None = None)#

Bases: Container

Tube container, like Eppendorf tubes.

Note that in regular use these will be automatically generated by the pylabrobot.resources.TubeRack class.
