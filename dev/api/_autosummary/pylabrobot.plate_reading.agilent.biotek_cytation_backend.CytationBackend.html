pylabrobot.plate_reading.agilent.biotek_cytation_backend.CytationBackend#

class pylabrobot.plate_reading.agilent.biotek_cytation_backend.CytationBackend(timeout: float = 20, device_id: str | None = None, imaging_config: CytationImagingConfig | None = None)#

Bases: BioTekPlateReaderBackend, ImagerBackend

Backend for Agilent BioTek Cytation plate readers.

The camera is interfaced using the Spinnaker SDK, and the camera used during development is the Point Grey Research Inc. Blackfly BFLY-U3-23S6M. This uses a Sony IMX249 sensor.
