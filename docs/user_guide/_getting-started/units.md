# Default Units

Unless otherwise specified, these are the units used in PyLabRobot:

| Quantity           | Unit                      |
| ------------------ | ------------------------- |
| Length             | millimeters (mm)          |
| Volume             | microliters (µL)          |
| Time               | seconds (s)               |
| Angle              | degrees (°)               |
| Weight/Mass        | grams (g)                 |
| Speed              | millimeters/second (mm/s) |
| Flow Rate          | microliters/second (µL/s) |
| Temperature        | degrees Celsius (°C)      |
