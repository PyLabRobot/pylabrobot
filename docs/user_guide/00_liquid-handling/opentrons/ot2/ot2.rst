Opentrons OT-2
==============

PyLabRobot supports the `Opentrons OT-2 <https://opentrons.com/ot-2/>`_ liquid handler. Single-channel pipettes (GEN1 and GEN2) on left and right mounts are supported.

Supported pipettes: ``p10_single``, ``p20_single_gen2``, ``p50_single``, ``p300_single``, ``p300_single_gen2``, ``p1000_single``, ``p1000_single_gen2``.
